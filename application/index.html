Application Overview

This API is developed using FastAPI, SQLAlchemy, Poetry, Redis, Celery, and python-dotenv for an AI-powered application that generates articles based on YouTube videos and other parameters.

Architecture

The application follows a microservice architecture with the following components:

        
          - app
              - main.py
              - auth_service
                - __init__.py
                - routes.py
                - models.py
                - controllers.py
              - user_service
                - __init__.py
                - routes.py
                - models.py
                - controllers.py
              - database
                - __init__.py
                - models.py
                - repository.py
              - config
                - __init__.py
                - settings.py
                - dotenv.py
              - docker-compose.yml
              - Dockerfile
              - requirements.txt
              - .env
  • main.py: The entry point of the application.
  • auth_service: Handles authentication-related functionality with its own routes, models, and controllers.
  • user_service: Manages user-related functionality with its own routes, models, and controllers.
  • database: Contains the database models and repository for interacting with the database.
  • config: Holds configuration files, including settings and dotenv management.
